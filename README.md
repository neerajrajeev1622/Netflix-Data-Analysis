# 🎬 Netflix Data Analysis Dashboard

## 📊 Overview

This project analyzes Netflix content using Tableau to uncover insights about content distribution, growth, genres, ratings, and cast impact.

---

## 🚀 Key Features

### 📌 Dashboard 1: Overview

* Total Content KPI
* Movies vs TV Shows comparison
* Content growth over years
* Genre distribution
* Rating distribution

### 📌 Dashboard 2: Content Insights

* Global content distribution (Map)
* Top countries analysis
* Duration analysis (Histogram)
* Content age analysis
* Cast size impact (Scatter Plot)

---

## 🛠️ Tools Used

* Tableau Public
* Excel (Data Cleaning)
